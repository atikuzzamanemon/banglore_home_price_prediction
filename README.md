Banglore Home Price Prediction:

This project predicts the price of a house in Bangalore based on location, square feet, number of bathrooms, and number of bedrooms (BHK).
It uses a Linear Regression machine learning model and a Flask API with a simple web interface.

Features: 

>> Predict home prices using basic inputs,
>> Trained ML model using real estate data,
>> Flask API for backend,
>> Simple HTML/JS frontend for predictions

Technologies: 

>>Python,
>> NumPy, Pandas, Scikit-Learn,
>> Flask,
>> Pickle,
>> HTML, CSS, JavaScript
